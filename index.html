Terva — harness the wildcard

An agent harness — a coding agent out of the box, open to anything you can wire a tool to. One static binary: one hardened core, many front ends, pluggable in any language. 

$ go install terva.sh/terva/cmd/terva@latest
github releases docs

features

One static binary

Download one file and run it. No runtime, no package manager, no daemon — Go all the way down.

Any provider

Anthropic, OpenAI, Gemini, Bedrock, Copilot, Groq, Ollama, and twenty more built in — plus any OpenAI-compatible endpoint.

Operate any tool

Extensions, MCP servers, and hooks in any language — JSON-RPC over subprocess — let terva drive services, hardware, or your own systems, not just files.

Chat connectors

A built-in Telegram bridge, and external connectors in any language over a small versioned JSON protocol.

Many front ends

One engine in your terminal, your editor (over ACP), and chat — plus -p print, --json, and a JSON-RPC server to embed it.

Consolidated & tested

One loop, one event wire, one policy — typed contracts, not parallel reimplementations — with golden and end-to-end harnesses backing every change.

usage

terva
$ terva                              # interactive tui
$ terva "fix the failing test"       # tui, pre-filled prompt
$ terva -p "list all go files"       # print final text, exit
$ terva --json "refactor main.go"    # newline-delimited json events
$ terva --continue                   # resume the last session here

install

go install
$ go install terva.sh/terva/cmd/terva@latest
prebuilt binaries

Every release ships archives for Linux, macOS, and Windows on amd64 and arm64, plus a checksums.txt. Download, verify, chmod +x, drop on your $PATH.

from source
$ git clone https://github.com/terva-sh/terva
$ cd terva
$ make build        # produces ./bin/terva

Then run terva and type /login — the TUI opens without credentials and walks you through browser-based auth: an API key or a Claude Pro/Max, ChatGPT Plus/Pro, Kimi Code, or GitHub Copilot subscription.

terva is Finnish for pine tar — the traditional preservative and cure-all that sealed boats and kept them seaworthy. Its agent persona is Mieli (MYEH-lee), Finnish for “mind”: a mind in a preserved vessel.

A hard fork of patriceckhart/zot that took its own name once it diverged too far to carry upstream's — focused on hardening the seams and widening the surface. zot continues upstream; terva is its own project alongside it — not a replacement, successor, or rename. Existing zot installs keep working unchanged; see the compat story.
